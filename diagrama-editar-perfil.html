🖥️ Frontend — Angular
UI Destaque de campos alterados
  • Label fica teal ao modificar
  • Badge "alterado" aparece no label
  • Borda do input muda de cor
  • isFieldChanged(field) em tempo real
  • isRotinasChanged() para picklist
Dialog Alterações não salvas
  • Ao fechar, verifica hasFormChanges()
  • Se houver mudanças → ConfirmDialog
  • "Sair sem salvar" ou "Continuar editando"
Guard Bloqueio systemDefault
  • Badge "🔒 Sistema" na tabela
  • Botões Editar e Excluir desabilitados
  • Toast de aviso se tentar abrir
  • Linha com opacidade reduzida
Confirm Confirmação de rotinas
  • Ao salvar com rotinas alteradas
  • "Alterar permissões afetará usuários
    vinculados a este perfil"
  • Aparece no form E no dialog de rotinas
Grid Colunas de data
  • Criado em: DD/MM/YYYY HH:MM
  • Atualizado em: DD/MM/YYYY HH:MM
  • Largura responsiva: 12% / min 9rem
  • formatDate() com Intl nativo
☕ Backend — Spring Boot
Validation UpdatePerfilRequest
  • code: @NotBlank · @Length(max=50)
  • name: @NotBlank · @Length(max=60)
  • icon: @Length(max=100) · opcional
  • active: Boolean · opcional
  • rotinaIds: Set<UUID> · opcional
Guard PerfilBloqueadoException
  • Lançada se systemDefault = true
  • E request tenta mudar code/name/active
  • Retorna HTTP 403 Forbidden
  • Atualizar rotinas é permitido
UseCase UpdatePerfilUseCase
  • Captura estado anterior (before)
  • Aplica mudanças (se não systemDefault)
  • Atualiza rotinas (todos os perfis)
  • Recarrega estado final (after)
  • Gera audit entries para cada diff
  • Objects.equals() — null safe
Audit Campos auditados
  • code — antes × depois
  • name — antes × depois
  • active — antes × depois
  • rotinas — IDs antes × IDs depois
  • Editor: SecurityContextHolder
  • Timestamp: Instant.now()
Migration V6 — Flyway SQL
  • ADD COLUMN system_default BOOLEAN
  • ADMIN e MANAGER → system_default = true
  • CREATE TABLE perfil_audit_log
  • FK: perfil_id → perfil(id)
  • INDEX em perfil_id, edited_at
🗄️ PostgreSQL — Schema dropshipping
Table perfil
  • id UUID PK
  • code VARCHAR(50) UNIQUE
  • name VARCHAR(60)
  • icon VARCHAR(100)
  • active BOOLEAN
  • system_default BOOLEAN ← novo
  • display_order INT
  • created_at / updated_at
Table perfil_rotina
  • perfil_id UUID FK
  • rotina_id UUID FK
  • Atualizada via replaceRotinasForPerfil
  • DELETE + INSERT por transação
Table perfil_audit_log ← nova
  • id UUID PK
  • perfil_id UUID FK
  • edited_by VARCHAR — quem
  • edited_at TIMESTAMP — quando
  • field_name VARCHAR — qual campo
  • value_before TEXT — valor anterior
  • value_after TEXT — valor novo
Regra Perfis protegidos
  • ADMIN → system_default = true
  • MANAGER → system_default = true
  • OPERATOR / SELLER → editáveis
  • Novos perfis → editáveis por padrão
